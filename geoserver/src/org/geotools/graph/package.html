Graph builder builds conencted line netwroks based on feature relationships.

This api follows the standard Builder Pattern for network generation, this may be customized by using your own FeatureComparator/FeatureRelator following the stratagy Pattern.

Once constructed you may access your Graph using a walker following the vistor pattern. Walkers exist for both edge and node based traversals.

The orgional graph package has been developed using the JUMP Feature api. Its intension is to be able to aid automated skeletonization of water bodies by computing directionality of generated skeletons. The work for the orgiional JUMP based library has been funded by the BC Proventional Government department of BMGS.

This library has been proted as part of the VWFS projection sponcered by Geo Innovations. If all goes well this will be included in a future release of geotools2.
