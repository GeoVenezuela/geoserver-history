GeoServer 1.7+ SQL Server Data Store Extension
==============================================

This package contains a SQL Server DataStore implementation that is
distributed as a separate plug-in.


Installation
------------

  1. Unzip this archive into the GeoServer library directory. In the binary
     install this is under '<GEOSERVER_ROOT>/webapps/geoserver/WEB-INF/lib'.

  2. Download the SQL Server JDBC driver from Microsoft:

        http://msdn.microsoft.com/en-us/data/aa937724.aspx

     Download version 1.2 of the driver (SQL Server 2005 driver)

  3. Copy the SQL Server JDBC driver into the GeoServer library directory.

  4. Restart GeoServer.


Compatability
-------------

This plugin will work with any GeoServer 1.7.x version.
