Project quickWMS: Generic JavaScript WMS Client
Author : Pedro Pereira Gonçalves
email : pedro.goncalves@esa.int or pedro@inovagis.org
Version : 0.3 (2003-11-18)

JavaScript classes for creating Web Map clients and interfacing
WMS servers according to OpenGIS Web Mapping Specification 
(versions 0.7 to 1.1). 
The goal of this project is to enable the fast creation of web 
mapping clients using javascript. The target browsers are 
Internet Explorer (version 5.5 and up) and Netscape/Mozilla
(7.00 and up) both for Windows, Mac and Linux.
