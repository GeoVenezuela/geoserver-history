.. _services:

Services
========
GeoServer serves data using standard protocols established by the `Open Geospatial Consortium <http://www.opengeospatial.org/>`_ (OGC). WCS allows for requests of coverage data (rasters); WFS allows for requests of geographical feature data (vectors); and WMS allows for requests of images generated from geographical data.  Within the Service Section of GeoServer, you can publishing of vector, raster and map data.
