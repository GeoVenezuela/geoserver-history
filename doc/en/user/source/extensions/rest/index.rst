.. _rest_extension:

REST
====

The REST extension provides a RESTful interface through which clients can 
configure a GeoServer instance through simple HTTP calls. With it clients can 
programatically configure the data served by GeoServer.

For information on getting and installing the REST extensions proceed to 
the :ref:`rest_installation` section. For details about the API and some 
hands on examples proceed to the :ref:`rest_config_api_ref` or :ref:`rest_config_examples` sections.
